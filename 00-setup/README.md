### 1. Install VS Code
Download and install [Visual Studio Code](https://code.visualstudio.com/). It's the industry standard editor.
**Recommended Extensions:**
- Prettier - Code formatter
- ESLint
- Live Server

### 2. Install Node.js
Download [Node.js (LTS Version)](https://nodejs.org/). This allows you to run JavaScript outside the browser.
Verify installation by opening your terminal and typing:
```bash
node -v
```

### 3. Your First Code
Create a file named `hello.js` and write:
```javascript
console.log("Hello, JavaScript!");
```
Run it in the terminal:
```bash
node hello.js
```
